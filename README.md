# OpenMix
PyTorch implementation of our CVPR2023 paper "OpenMix: Exploring Out-of-Distribution samples for Misclassification Detection"

### Usage 
We run the code with torch version: 1.10.0, python version: 3.9.7
```
python main_cvpr.py
```
